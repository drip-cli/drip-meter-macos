# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. By participating in this project — issues,
pull requests, discussions, or any other space the project owns —
you agree to abide by its terms.

The full text is available at the link above.

## Reporting

If you see behaviour that violates the Covenant, open a confidential
issue on the GitHub repository or email the maintainers directly.
Reports are reviewed promptly and handled discreetly.

## Scope

The Covenant applies to all project spaces and any time someone is
representing the project in public.
